函式 Function - 參數 part.1

函式 Function - 參數 part.1

  1. 參數 ( parameter )
  2. 位置引數( Position Argument )

參數( parameter )

使用print( )函式，底下的程式碼會印出hello：

text = 'hello'
print(text)

我們是將變數text作為參數傳遞進去print( )函式之中！所以直譯器的結果會顯示hello。
所以我們這裡可以得知函式的另一種樣貌：

def 函式名稱( 參數1 , 參數2....):
    ....參數1....
    ...執行主體...

參數可有可無，並沒有數量的限制，但無倫是否有參數都需要保留小括弧( )！

如果有多個參數，Python以逗號做為區隔，並以位置引數或關鍵字引數來做參數的對應。

範例：

傳遞參數範例-1

def helloPeople(name):
    print('hello ' + name)

name = 'Roy'
helloPeople(name)

位置引數( Position Argument )

函式的參數以位置引數來做為對應依據。

如果傳遞參數時的順序不對，那對應就會發生錯誤。

課堂練習：

位置引數練習-1

#請修正位置引數，傳遞正確的參數
def show(name, age, gender):
    print('姓名為' + name)
    print('年齡為' + age)
    print('性別為' + gender)

show('Austin','Boy','13')

位置引數練習-2

長方體表面積=（長×寬+長×高+寬×高）×2
請讓使用者可以輸入長方體的長、寬、高，
傳入一個SurfaceArea()函式，印出長方體的表面積！

測試案例：
長：10，寬：2，高：7
表面積為208

位置引數練習-3

攝氏溫度的轉成公式為：
華氏溫度 = 9/5*攝氏溫度 + 32
絕對溫度 = 攝氏溫度 + 273.2

請讓使用者輸入攝氏溫度，
並讓使用者選擇要轉換成哪種溫度，1:華氏溫度、2:絕對溫度
最後印出轉換後的結果！

提示：
請練習將功能與流程分開撰寫！

測試案例-1：
攝氏溫度:100
選擇1:華氏溫度、選擇2:絕對溫度
1
等於華氏溫度212度

測試案例-2：
攝氏溫度:100
選擇1:華氏溫度、選擇2:絕對溫度
2
等於絕對溫度373.2K
