🗒️命令提示字元指令

  1. 【dir】查詢目錄：列出所處位置上的所有檔案名稱。

  2. 【cd】【chdir】變更目錄：在所處位置移動到其子資料夾中。
    可以使用tab鍵，在資料夾間快速切換，不必打完完整的資料夾名稱。

  3. 【md】【mkdir】建立目錄：加上檔案名稱，即可在所處位置建立資料夾。
    mkdir newFolder

  4. 【cls】清除視窗：清除命令提示字元的歷史紀錄。

  5. 【start】執行程式：加上 應用程式名稱 與 檔案(或網址) 名稱，即可開啟檔案。
    start chrome https://google.com
    start word 範例文字檔.txt

🗒️執行Python程式

  1. 現在有一個 python檔 被放置在桌面資料夾【Desktop】裡。

  2. 點選左下角的【windows開始按鈕】，在搜尋欄中輸入並開啟【命令提示字元】或【CMD】

  3. 使用【變更目錄指令】cd Desktop，
    從原本預設的路徑【C:\Users\使用者名稱】，
    移動到python程式碼檔所屬的桌面資料夾【Desktop】中。

    按下Enter後，路徑位置已經移動到【C:\Users\使用者名稱\Desktop】裡了。

  4. 使用【查詢目錄指令】dir，
    列出在【Desktop】資料夾裡的所有檔案，看看是不是有【test.py】？

  5. 確定test.py檔案確定存在後，
    使用【執行指令 + 檔案名稱】執行python檔。

    python test.py

  6. 成功執行python檔案

🗒️補充 - 為什麼要添加環境變數呢？

📝簡單來說，就是跟系統說【我要執行的指令，它的執行檔位置在這裡喔】。

作業系統會依照環境變數中所設定的路徑順序，依序尋找各路徑下是否有這個指令。
查看目前電腦中的環境變數：
1️⃣打開命令提示字元
2️⃣輸入echo %PATH%
3️⃣

📝當你直接輸入一個沒有添加位置到環境變數的指令時，系統並不知道要去哪裡執行這項指令。
就算我們安裝已經將Python安裝到了系統之中，它就和大部分的應用程式一樣，靜靜地待在C:\內；如果要使用它，就必須輸入完整的路徑，好讓電腦執行。

📝在正常情況下，Python執行檔的所在位置為C:\Users\使用者名稱\AppData\Local\Programs\Python\Python36-32\python.exe
要用Python指令執行.py檔就必須輸入"C:\Users\使用者名稱\AppData\Local\Programs\Python\Python36-32\python.exe" 檔案名稱.py

📝把這個路徑添加到環境變數之後，系統認識了Python執行檔的位置，就可以使用python 檔案名稱.py，這麼簡潔的指令執行了。
