函式 Function - return指令

當我們呼叫一個函式，在執行完畢之前，程式的焦點都會停留在函式內；但有時候我們需要跳出函式、將焦點轉回流程之中。

或者，我們需要將一個函式的運算結果、傳遞到另外一個函式再進一步計算。

以上這兩種情況，我們都需要用到return指令。

  1. 提前離開函式
  2. 回傳值
  3. 回傳值定義
  4. 回傳值跳脫
  5. 回傳特殊型別

提前離開函式

如果在函式中有多個【if判斷式】作流程控制，可以用作【如果不符合判斷時】跳脫函式的指令。

課堂練習：

return指令練習-1

#請完成以下函式：
#傳進example()的參數如果大於100，則跳出函式。
#如果不大於100，則判斷參數是否為偶數，並印出"是偶數"

def example(num):
    print('是偶數')

回傳值

像是能【回傳】字串或串列長度的【len( )函式】，就是一個擁有return指令的函式。回顧字串函式與方法。

也因為它會回傳資料，所以我們可以使用一個變數去儲存它的回傳值。

#一個串列儲存四筆資料
list1 = [1, 2, 3, 4]

#使用變數length去儲存len()函式的回傳值
length = len(list1)

#印出length可以得到 4
print(length)

回傳值定義

如果要自己定義回傳值，只要使用return指令，其後再加上回傳值即可。

【第7行】：一個函式可以有多個回傳值，只需以逗點隔開即可。

【第9行】：如果有函式多個回傳值，則需使用同等多的變數來承接它們，一樣以逗點隔開即可。

1. def example(參數1, 參數2):
2. 
3.     .............
4.     ...執行主體...
5.     .............
6.
7.     return 回傳值1, 回傳值2
8.
9. var1, var2 = example(100,200)
課堂練習：

return指令練習-2 / 解答

#寫一個函式讓使用者能輸入"被除數"與"除數"，
#而函式則會回傳"商"與"餘數"。
#最後將其印出。

測試案例：

被除數: 50
除數: 6
50 除以 6 等於 8 餘 2

回傳值跳脫

而一旦執行一個return指令，就會馬上脫離當前的函式，並不會持續回傳不同的值。

範例：

return指令範例-1

def example(num1, num2):

    if num1 > 10:
        return num1
    if num2 > 10:
        return num2

回傳特殊型別

回傳值可以有多種型別，可以是字串、數值、布林值、函式等，只要是被定義在函式內的物件都可以。

課堂練習：

return指令練習-3 / 解答

#寫一個combine()函式將五個數值合併成一個串列、並回傳這個串列給使用者。
combine(1,2,3,4,5)
