畫出圓形與橢圓形

畫出圓形

Turtle模組的第二個挑戰，控制小烏龜爬出圓形！

  1. 創建python檔

    打開sublime3， Ctrl + N 開啟一個新檔案， 緊接著Ctrl + S存檔， 將檔案命名為turtle-square.py， 並 且將儲存目的地選在"桌面"。(方便待會兒操作)

  2. 匯入所需模組

    和所有模組一樣，首先使用import將turtle模組匯入程式之中：

    import turtle
  3. 操作指令

    首先，讓我們使用指令showturtle()顯示小烏龜的位置，會發現它位在畫面的正中心( 0 , 0 )，並且鼻尖向右。

    #使用指令showturtle()來顯示小烏龜
turtle.showturtle()

  4. 圓型指令

    找到指令circle(r,extent,step)，繪製一個指定半徑r、弧度範圍extent、階數step(正多邊形)的圓。
    我們這裡只需要指定半徑就好，其餘的兩個參數可以不填。
    將 r 設定為100。

    #使用指令circle()，輸入參數100，讓小烏龜爬出一個圓。
turtle.circle(100)

👽挑戰題

🍰蛋糕🍰

請嘗試使用circle( )的另外兩個參數，搭配forward( )等指令，讓小烏龜爬出一個6分之一圓的蛋糕形狀吧！

畫出橢圓形

Turtle模組的第三個挑戰，控制小烏龜印出橢圓形！
在電腦中畫出橢圓形一直都是最難的挑戰，硬解公式並不是最好的做好。

所以這次我們來改變畫筆的形狀，將畫筆改為圓型、並且改變長寬比，最後再用stamp()指令來蓋章。

  1. 創建python檔

    打開sublime3， Ctrl + N 開啟一個新檔案， 緊接著Ctrl + S存檔， 將檔案命名為turtle-square.py， 並 且將儲存目的地選在"桌面"。(方便待會兒操作)

  2. 匯入所需模組

    和所有模組一樣，首先使用import將turtle模組匯入程式之中：

    import turtle
  3. 操作指令

    首先，讓我們使用指令showturtle()顯示小烏龜的位置，會發現它位在畫面的正中心( 0 , 0 )，並且鼻尖向右。

    #使用指令showturtle()來顯示小烏龜
turtle.showturtle()

  4. 修改畫筆

    使用shape()指令，改變畫筆形狀，輸入參數circle。
    使用shapesize()指令，改變圓型的長寬比，這裡輸入參數(10,15)。
    10:15，因此可以預測這是一個扁扁的橢圓形。

    turtle.shape("circle")
turtle.shapesize(10,15)
