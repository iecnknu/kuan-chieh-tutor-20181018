  2. 選擇性引數

關鍵字引數( Keyword Argument )

若一個函式有多個參數要傳遞處理，使用位置引數的方式來傳遞參數常常會搞混或錯位，這時候關鍵字引數就派上用場了！

直接呼叫關鍵字來對應要傳入的參數。

課堂練習：

關鍵字引數練習-1

#看看錯誤提示是什麼，並找出錯誤之處。
def show(name, age, gender, school, father, mother):
  print('姓名為' + name)
  print('年齡為' + age)
  print('性別為' + gender)
  print('學校為' + school)
  print('爸爸為' + father)
  print('媽媽為' + mother)

show(gender = 'Boy',name = 'Austin','13')

選擇性引數( Optional Argument ) / 預設引數值

定義函式時，可以預設要傳遞進去的參數，我們稱之為選擇性引數。

定義函式時，選擇性引數必須放在一般引數的後面。

若函式有預設引述值，在呼叫該函式時可不用填寫有預設引述值的參數，函式會自動使用預設引述值帶入。

範例：

選擇性引數範例-1

到麥當勞買套餐，A餐預設就會是"中杯可樂+中薯"，但同時仍然能客製化客人的需求。
請寫一個A_Meal函式，讓櫃台人員能輕易幫客人點餐。

def A_Meal(drink='中杯可樂',Frise='中薯'):
  print(drink + ' 和 ' + Frise)
