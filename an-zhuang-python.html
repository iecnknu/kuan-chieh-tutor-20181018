Python

  1. 官方網站簡介

https://www.python.org/

🗒️官方網站簡介

  1. 滑鼠移至 文件(Documentation)，選擇Python 3.X 觀看Python 官方文件
    雖然過時了，但是python 2.7仍然為許多平台的主力。

  2. 右上方能選擇【語言】及【python版本】
    教學範例【Tutorial】詳細描述了Python的所有一切，所有的問題都可以在這裡得到解答。

安裝教學

  1. 滑鼠移至 下載(Downloads)，選擇最新的Python 3.6.X
    點選之後會開始下載。

  2. 下載完成後，點開【python-3.6.4.exe檔】，選擇執行

  3. 一定要勾選下方條件，【Add Python 3.6 to PATH】(紅色部分)，再點選上方【Install Now】(藍色部分)
    環境變數是在告訴電腦，當我們下達Python指令時，它可以到哪個路徑去尋找Python.exe以便運行程式。

  4. 安裝進行中

  5. 待進度條完成後，即可將視窗關閉。到這裡，Python已經順利地安裝到電腦上了！

  6. 檢查Python是否完整安裝：
    點選左下角的【windows開始按鈕】，在搜尋欄中輸入並開啟【命令提示字元】或【CMD】

  7. 開啟命令提示字元後，輸入

    python -V

    -V 代表 version，電腦會回傳安裝的python版本給你。

    版本顯示正確後，再次輸入

    python

    即可進到Python直譯環境

如果以上步驟皆顯示正確，就代表Python在電腦上安裝成功了！
