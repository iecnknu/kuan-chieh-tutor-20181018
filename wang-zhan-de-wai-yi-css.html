🗒️什麼是CSS
  • CSS，Cascading Style Sheets階層式樣式表
  • 名字裡有【Style樣式】，顧名思義就是要用來定義HTML5的元件在瀏覽器上呈現的樣子。

  • 在先前的HTML5標籤的章節中，其實我們已經透過【style屬性】在標籤中設置過CSS了。

    style="color:#4285F4"

    但先前設置的CSS，只能影響到該標籤範圍的，如果要套用到別的標籤範圍就需要重新撰寫一次，不僅佔空間也費時費力。

🗒️CSS的組成

CSS 規則由兩個主要的部分構成：【選擇器selector 】，以及一個或多個【屬性property】(使用分號間隔)。

選擇器 { 屬性1 : 值 ; 屬性2 : 值 }

更為好辨認的寫法為：

選擇器 {
    屬性1 : 值 ; 
    屬性2 : 值
}
🗒️CSS的選擇器

選擇器就是用來選擇要對哪一些【tag標籤】、【class類別】或【id】套用CSS樣式。

  • 選擇【tag標籤】的話，直接撰寫tag名稱即可： 這樣一來標籤是body的都會套用這個CSS。
body {
 text-align : center ;
 color : green 
}
  • 選擇【class類別】的話，需要在名稱前加上【.】： 這樣一來類別是sky的標籤都會套用這個CSS。
.sky {
 text-align : left ;
 color : blue 
}
  • 選擇【id】的話，需要在名稱前加上【#】： 這樣一來id是hahaha的標籤都會套用這個CSS。
#hahaha {
text-align : right
color : red
}

這樣一來，同樣顯示效果的CSS只要撰寫一次，卻可以透過選擇器指定給多個不同的內容使用。

🗒️<style>標籤的建立

我們可以在<head>標籤裡建立一個<style>標籤，把所有的CSS樣式都放在裡面。

<!DOCTYPE html>
<html>
    <head>
        <title>CSS範例</title>
        <meta charset="utf-8">
        <style type="text/css">

            body {
                text-align : center ;
                color : green
                }

            .sky {
                text-align : left ;
                color : blue
            }

            #hahaha {
                text-align : right ;
                color : red
            }

        </style>
    </head>
    <body>
        <div>這個會顯示綠色</div>
        <div>這個會偏左，顯示藍色</div>
        <div>這個偏右，顯示紅色</div>
    </body>
</html>
👀動手試試看

  1. 依照上述每個<div>內文字的要求，觀察<head>裡<style>的CSS，替這些<div>套用適合的【類別】或【id】。

  2. 如果我在body標籤中新增底下這行，會怎麼顯示呢？

     <span>你好</span>
🗒️CSS文件的建立

雖然把CSS放置在<head>中，也使用學會選擇器，控制樣式的靈活度已經大大增加。
但一個網站通常不會只有一個頁面(一個html文件)，而且幾乎都會共享同一種主題風格。
如果需要在每個html文件的<head>中，都插入這麼一大串CSS樣式，不僅會造成伺服器的負擔，工程師也不好維護或修改。

所以外置的樣式表，也就是獨立的CSS文件就應運而生了。

👀創建CSS檔

  1. 打開Sublime3，
    Ctrl + N 開啟一個新檔案，
    緊接著Ctrl + S存檔，
    將檔案命名為Mycss.css，
    並且將儲存目的地選在"桌面"。(方便待會兒操作)

  2. 新增底下程式到文件中

     body {
         text-align : center ;
         color : green
     }

 .sky {
         text-align : left ;
         color : blue
     }

 #hahaha {
         text-align : right ;
         color : red
     }

  3. 〈link 〉連線標籤
    開發者只要在伺服器上佈署好檔案的位置，並且html文件的<head>標籤中，使用link標籤，指定連線到該CSS檔案，瀏覽器便知道該HTML文件要套用哪個CSS。

<link rel="stylesheet" type="text/css" href="MyCSS.css">
🗒️補充：Google Fonts

Google Fonts字體庫，使用方式和CSS很像，它提供了線上的字體庫給瀏覽器使用。
即使使用者沒有安裝該字體在電腦上，仍然能以各式各樣的字體來顯示。

👀動手做做看

請結合模擬的Google搜尋頁面的html文件，將CSS分類並建立獨立的檔案，同時引用Google Fonts，將文件內所有的字體都設置成Roboto。
最後，請到W3School【點我觀看文字的屬性】，查看如何改變字體的大小，並將Google幾個字設定為50px。
