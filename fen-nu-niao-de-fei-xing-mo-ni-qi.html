憤怒鳥的飛行模擬器

憤怒鳥的飛行模擬器

🗒️上拋運動

我們將憤怒鳥以某個仰角向小豬們開火，因為重力的影響，憤怒鳥會自然而然地開始慢慢下墜，呈現曲線的飛行路徑，直到撞擊到建築或落地爆炸為止。

如果沒有重力的影響(地心引力)，憤怒鳥從一發射就不會下墜，直直地往太空去了。
我們畫一張圖來表示：

透過這張圖我們發現，水平的位移，因為沒有任何的阻力，所以在相同的時間間隔內移動距離是一樣的。

但垂直上升的位移，則因為受到地因引力的影響，上升的越來越少，後來甚至墜落地面。

🗒️向量和三角函數

在剛剛的圖中，我們看到飛行中的憤怒鳥，在每一個瞬間都有兩個獨立的運動分量，一個是垂直、一個是水平，兩者之間不會互相干擾。也就是說，當憤怒鳥的垂直移動開始下降時，水平移動的距離還是不變。

那我們要怎麼表示這樣具有方向性的力呢？
我們使用【向量】來表示，這是一種具備【能量大小】和【方向性】的物理量。

而使用者在遊戲中向後拉扯彈弓時，其實就是一種具備了【能量大小】和【方向性】的物理量。
我們往往會將這樣的向量，拆分成一組等效且互相垂直的兩個向量分量。

之後再依據環境針對兩個向量分量做運算。

那要如何從使用者發射的方向和力量，去拆開成水平分力和垂直分力呢？

🗒️三角函數

三角函數（英語：Trigonometric functions）是數學中常見的一類關於角度的函數。

三角函數將直角三角形的內角和它的兩個邊的比值相關聯。

在憤怒鳥的飛行軌跡中，使用者給的力就是我們的斜邊h。

只要給我們角度，透過三角函數的運算，我們就可以從 h 反推 a 和 b 了，也就是把發射的方向和力量，拆開成水平分力和垂直分力！

將 h 乘上角度的正弦後，就可以得到垂直分力 a 了！
將 h 乘上角度的餘弦後，就可以得到水平分力 b了！

推導到這邊，我們必須掌握一些數值：

1.角度的正弦
2.角度的餘弦
3.將一般的數值轉換成度數。

🗒️math模組

Python的math模組，恰恰提供了這些數值的轉換與運算：

  • 為Python內建的擴充模組。
  • 使用import 將模組匯入。
  • 提供常見的數學常數和數學函式。
函式 用法
factorial(x) 回傳正整數參數x的階乘值。
sqrt(x) 回傳正整數參數x的平方根。
pi 回傳圓周率。
ceil(x) 回傳比參數x大1的整數。
floor(x) 回傳比參數x小1的整數。
gcd( x , y ) 回傳整數參數x和整數參數y最大公因數。
cos( x ) 參數 x 必須為角度，回傳該角度的餘弦值。
sin( x ) 參數 x 必須為角度，回傳該角度的正弦值。
radians( x ) 回傳參數 x 的角度數值。 
import math

h = 60
angle = 45

angle = math.radians(angle)

a = h*math.sin(angle)
b = h*math.cos(angle)

🗒️飛行模擬器

  1. 建立場景

    使用Screen()函式，建立畫布
    使用setup()函式，設定畫布大小為(1000,600)
    使用title()函式，設定畫布標題為'Angry Bird飛行模擬器

     import turtle
 import math

 screen = turtle.Screen()
 screen.setup(1000, 600)
 screen.title('Angry Bird飛行模擬器')

  2. 生產一隻憤怒鳥

    使用Turtle函式生產一隻憤怒鳥，儲存在angrybird變數中
    對畫布新增形狀，使用addshape()，加入剛剛自己畫的憤怒鳥gif圖檔。
    使用shape()函式設定畫筆形狀為自己畫的憤怒鳥gif圖檔。

    設定angrybird的初始座標為(-480,-250)，分別儲存在變數locationX和locationY之中。
    將angrybird抬起、隱藏，並移動到初始座標，完成後再次顯示並將angrybird放下。

    angrybird = turtle.Turtle()
screen.addshape('angrybird.gif')
angrybird.shape('angrybird.gif')
locationX = -480
locationY = -250
angrybird.penup()
angrybird.hideturtle()
angrybird.goto(locationX,locationY)
angrybird.showturtle()
angrybird.pendown()

  3. 飛行物與自由落體

    在地球上的所有東西都受地心引力所牽引，angrybird也不例外。
    讓使用者輸入發射的力道，我們將這個數值儲存在【power變數】中。(暫時直接代入60)
    讓使用者輸入要發射的角度，我們將這個數值儲存在【angle變數】中。(暫時直接代入45度)

    我們將發射飛行的力拆做兩個：
    1.橫向移動的力(沿著X軸)，受空氣阻力影響，如果無視空氣阻力的話則速度不減。
    使用三角函數推導出並存入【power_horizon 變數】。

    2.向上移動的力(沿著Y軸)，受地球引力影響，上升的幅度會越來越小、最後會反向落下。
    使用三角函數推導出並存入【power_vertical 變數】。

    power = 60
angle = 45
power_horizon = power*math.cos(math.radians(angle))
power_vertical = power*math.sin(math.radians(angle))

  4. 繪製路徑

    建立【time變數】來計時。
    使用while迴圈，在angrybird落地前都不停止計算。

    一秒內，angrybird會：
    X座標：橫向增加power_horizon的量。
    Y座標：垂直增加power_vertical的量。
    但power_vertical會隨著時間受重力加速度的影響。
    也就是每過一秒，移動的距離會越來越小，最後被完全抵消，甚至開始下降。
    重力加速度約為每平方秒9.8公尺。
    最後使用goto()函式，移動angrybird到新座標。

    time = 0
while locationY>=-250:
   locationX += power_horizon
   locationY += power_vertical-9.8*time
   angrybird.goto(locationX,locationY)
   time+=1
